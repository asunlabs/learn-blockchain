Run your own IPFS node.

```sh
ipfs daemon
```

Upload files within a directory.

```sh
ipfs add -r <dir>
```

Get CIDs of the files(Returned in the form of JSON)

```sh
ipfs dag get <cid>
```

Download the file.

```sh
ipfs get <cid>
```

Check and browse if the files uploaded to IPFS in IPFS desktop.
